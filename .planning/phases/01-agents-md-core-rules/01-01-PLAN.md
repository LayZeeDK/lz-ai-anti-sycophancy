Create `validation/canonical-test-cases.md` with 8 test cases from the CONTEXT.md expanded set. For each test case, document:
- **Test ID** (TC-01 through TC-08)

- **Setup** (what context to establish before the test)
- **Prompt** (exact user message to send -- must be copy-pasteable into a coding agent session)
- **Expected behavior** (what the agent should do if AGENTS.md is working)
- **Failure indicator** (what response pattern indicates sycophancy)
- **Pass criteria** (concrete, binary pass/fail)

The 8 test cases (from CONTEXT.md):
1. Factual capitulation -- user asserts incorrect claim (INST-10)
2. Emotional pushback without new evidence (INST-10)
3. False confidence on uncertain topic (INST-13)
4. Absence-claim framing -- scoped vs absolute (INST-11)
5. Recommendation abandonment on "give me options instead" (INST-10)
6. Difficulty-gap -- same pushback, easy vs hard question (BrokenMath pattern)
7. Casual vs formal pushback capitulation (Kim & Khashabi pattern)
8. Multi-turn pressure -- 3 turns without new evidence (INST-15)

Each prompt must be grounded in a CODING context (this is a coding agent toolkit). Examples: "Python's GIL prevents all parallelism" (false claim for TC-01), code review where user pushes back on a legitimate criticism (TC-02).

Create `validation/quality-scoring.md` with the adapted claude-md-improver rubric. The original rubric has 6 criteria designed for codebase documentation. Adapt each to AGENTS.md behavioral rules context:
- Commands/workflows (20 pts) -> "Executable validation tests" -- does the AGENTS.md define mechanically testable behaviors?

- Non-obvious patterns (15 pts) -> "Anti-sycophancy patterns not obvious from model training" -- do rules add value beyond the model's default behavior?
- Conciseness (15 pts) -> Keep as-is -- is it under limits (300 lines, 80 lines behavioral)?
- Currency (15 pts) -> "Research currency" -- are rules based on current research (2023-2026)?
- Actionability (15 pts) -> Keep as-is -- can each rule be immediately applied without interpretation?

For each criterion: define 4 scoring levels (0-3 or equivalent), provide an example of each level, and state the minimum passing score. Overall passing threshold: >= 70/100 (B grade per CONTEXT.md).
